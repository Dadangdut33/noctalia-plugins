# Noctalia Plugins

My own collections of Noctalia plugins.

## Included Plugins

### `niri-focus-arrows`
A Noctalia Bar Widget for Niri that adds left/right focus controls.

- Click arrows to move focus between Niri columns
- Scroll over the widget to move left/right
- Optional wrap-around column movement
- Optional combined mode (single control with left/right click actions)

Plugin folder: `niri-focus-arrows/`

## Install

1. Add this repository as a custom plugin source in Noctalia.
2. Find the plugin you want to install and enable it.
